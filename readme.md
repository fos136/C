🌈 1. 무지개 사각형을 그리는 코드
기존 코드는 삼각형을 그리는 코드입니다. 정점이 3개만 정의되어 있습니다:


float v[] = {
    -0.5f, -0.5f, 1, 0, 0,
     0.5f, -0.5f, 0, 1, 0,
     0.0f,  0.5f, 0, 0, 1,
};

glDrawArrays(GL_TRIANGLES, 0, 3);
사각형을 그리기 위해서는 두 개의 삼각형을 사용해야 하므로 정점 6개가 필요합니다.
다음과 같이 코드를 수정합니다:


float v[] = {
    -0.5f, -0.5f, 1, 0, 0,
     0.5f, -0.5f, 0, 1, 0,
    -0.5f,  0.5f, 0, 0, 1,

     0.5f, -0.5f, 0, 1, 0,
     0.5f,  0.5f, 1, 1, 0,
    -0.5f,  0.5f, 0, 0, 1,
};

glDrawArrays(GL_TRIANGLES, 0, 6);
✅ 이처럼 정점 데이터를 6개로 바꾸면 무지개 색상의 사각형을 그릴 수 있습니다.

🫖 2. 와이어 프레임으로 회전하는 주전자
아래는 기존 코드입니다:

cpp
복사
편집
glColor3f(0.8f, 0.2f, 0.2f);
glutSolidTeapot(0.5);
glColor3f(...)는 주전자의 색상을 지정합니다.

glutSolidTeapot(...)은 실체(솔리드) 형태의 주전자를 그립니다.

이를 회전하는 와이어 프레임 주전자로 바꾸려면 아래와 같이 수정합니다:

glColor3f(0.0f, 1.0f, 0.0f);      // 초록색 설정
glutWireTeapot(0.5);             // 와이어프레임 주전자
✅ 이렇게 수정하면 초록색 와이어 프레임 주전자가 회전하며 렌더링됩니다.
